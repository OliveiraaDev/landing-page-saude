🏥 Landing Page – Área da Saúde

Este projeto é uma landing page profissional desenvolvida para clínicas e profissionais de saúde que desejam captar mais pacientes e aumentar sua presença digital.

A página foi construída com foco em:

✅ Design moderno e responsivo (mobile-first)

✅ Estrutura otimizada para conversão

✅ Seções estratégicas (sobre, serviços, depoimentos, formulário de contato)

✅ Código limpo e organizado em HTML, CSS e JavaScript

🚀 Tecnologias Utilizadas

HTML5 – Estrutura da página

CSS3 – Estilização responsiva e animações

JavaScript (Vanilla) – Interatividade e validações

(opcional: Bootstrap/Tailwind se usar framework de CSS)

📌 Estrutura da Página

Header (Topo) – Logo + menu fixo

Hero Section – Chamada principal com botão de agendamento

Sobre a Clínica – Breve apresentação

Serviços – Lista clara com ícones

Depoimentos – Prova social para gerar confiança

Formulário de Contato – Captura de leads (nome, e-mail, telefone)

Rodapé (Footer) – Links úteis e informações de contato
